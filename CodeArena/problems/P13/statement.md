# Minimum Window Substring

## Description
Étant données deux chaînes `s` et `t`, trouvez la plus petite sous-chaîne de `s` qui contient tous les caractères de `t` (avec leurs fréquences).

## Format de l'entrée
- La première ligne contient la chaîne `s` (1 ≤ |s| ≤ 100000)
- La deuxième ligne contient la chaîne `t` (1 ≤ |t| ≤ 100000)

## Format de la sortie
Afficher la plus petite fenêtre. S'il n'y en a pas, afficher "-1".

## Contraintes
- 1 ≤ |s|, |t| ≤ 100000
- Les chaînes contiennent des lettres majuscules et minuscules
- Time limit: 2 secondes
- Memory limit: 256 MB

## Exemple d'entrée
ADOBECODEBANC
ABC

## Exemple de sortie
BANC
