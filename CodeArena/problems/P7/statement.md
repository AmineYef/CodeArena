# Binary Search

## Description
Étant donné un tableau trié et une valeur cible, déterminez si la valeur existe dans le tableau.

Si elle existe, retournez son indice (en commençant à 1). Sinon, retournez -1.

## Format de l'entrée
- La première ligne contient deux entiers `n` et `target` (1 ≤ n ≤ 1000000)
- La deuxième ligne contient `n` entiers triés par ordre croissant (-10⁹ ≤ aᵢ ≤ 10⁹)

## Format de la sortie
Afficher l'indice de la cible (1-indexed) ou -1 si elle n'existe pas.

## Contraintes
- 1 ≤ n ≤ 1000000
- -10⁹ ≤ aᵢ, target ≤ 10⁹
- Le tableau est trié
- Time limit: 2 secondes
- Memory limit: 256 MB

## Exemple d'entrée
5 6
1 3 5 6 9

## Exemple de sortie
4
