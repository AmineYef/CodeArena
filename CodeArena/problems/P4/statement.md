# Missing Number

## Description
Vous avez un tableau contenant n-1 nombres distincts pris dans l'intervalle [1, n].

Trouvez le nombre manquant.

## Format de l'entrée
- La première ligne contient un entier `n` (2 ≤ n ≤ 1000000)
- La deuxième ligne contient `n-1` entiers distincts de [1, n]

## Format de la sortie
Afficher le nombre manquant.

## Contraintes
- 2 ≤ n ≤ 1000000
- Tous les nombres sont distincts et dans [1, n]
- Time limit: 2 secondes
- Memory limit: 256 MB

## Exemple d'entrée
5
1 2 4 5

## Exemple de sortie
3
