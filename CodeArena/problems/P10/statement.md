# Kth Largest Element

## Description
Trouvez le k-ième plus grand élément dans un tableau non trié.

## Format de l'entrée
- La première ligne contient deux entiers `n` et `k` (1 ≤ k ≤ n ≤ 100000)
- La deuxième ligne contient `n` entiers (1 ≤ aᵢ ≤ 10⁹)

## Format de la sortie
Afficher le k-ième plus grand élément.

## Contraintes
- 1 ≤ k ≤ n ≤ 100000
- 1 ≤ aᵢ ≤ 10⁹
- Time limit: 2 secondes
- Memory limit: 256 MB

## Exemple d'entrée
6 2
3 2 1 5 6 4

## Exemple de sortie
5

## Explication
Les éléments triés: [6, 5, 4, 3, 2, 1]. Le 2ème plus grand est 5.
