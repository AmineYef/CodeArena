# Palindrome Check

## Description
Déterminez si une chaîne donnée est un palindrome.

Un palindrome est un mot qui se lit de la même façon de gauche à droite et de droite à gauche (en ignorant les espaces, la ponctuation et la casse).

## Format de l'entrée
- Une seule ligne contenant la chaîne `s` (1 ≤ |s| ≤ 100000)

## Format de la sortie
Afficher "YES" si la chaîne est un palindrome, "NO" sinon.

## Contraintes
- 1 ≤ |s| ≤ 100000
- La chaîne peut contenir des lettres, chiffres, espaces et ponctuation
- Time limit: 2 secondes
- Memory limit: 256 MB

## Exemple d'entrée
A man a plan a canal Panama

## Exemple de sortie
YES
