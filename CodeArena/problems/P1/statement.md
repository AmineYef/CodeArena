# Two Sum

## Description
Étant donné un tableau d'entiers et une valeur cible, trouvez deux nombres dans le tableau dont la somme est égale à la cible.

Vous devez retourner les indices (positions) de ces deux nombres. Les indices commencent à 1.

## Format de l'entrée
- La première ligne contient deux entiers `n` et `target` (2 ≤ n ≤ 100000, -10⁹ ≤ target ≤ 10⁹)
- La deuxième ligne contient `n` entiers `a₁, a₂, ..., aₙ` (-10⁹ ≤ aᵢ ≤ 10⁹)

## Format de la sortie
Afficher deux entiers — les indices des deux nombres (le plus petit indice en premier).
Il est garanti qu'une solution existe et qu'elle est unique.

## Contraintes
- 2 ≤ n ≤ 100000
- -10⁹ ≤ aᵢ, target ≤ 10⁹
- Time limit: 2 secondes
- Memory limit: 256 MB

## Exemple d'entrée
4 9
2 7 11 15

## Exemple de sortie
1 2
