# Valid Parentheses

## Description
Déterminez si une chaîne de parenthèses est valide.

Une chaîne de parenthèses est valide si:
- Chaque parenthèse ouvrante a une parenthèse fermante correspondante
- Les parenthèses sont correctement imbriquées

Types de parenthèses: (), [], {}

## Format de l'entrée
- Une seule ligne contenant la chaîne `s` (1 ≤ |s| ≤ 100000)
- La chaîne contient uniquement les caractères: ( ) [ ] { }

## Format de la sortie
Afficher "YES" si la chaîne est valide, "NO" sinon.

## Contraintes
- 1 ≤ |s| ≤ 100000
- Time limit: 2 secondes
- Memory limit: 256 MB

## Exemple d'entrée
{[()]}

## Exemple de sortie
YES
