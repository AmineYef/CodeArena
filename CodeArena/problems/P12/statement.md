# Number of Islands

## Description
Étant donnée une grille 2D de '1' (terre) et '0' (eau), comptez le nombre d'îles.

Une île est entourée d'eau et formée en connectant des terres adjacentes horizontalement ou verticalement.

## Format de l'entrée
- La première ligne contient deux entiers `n` et `m` (1 ≤ n, m ≤ 300)
- Les `n` lignes suivantes contiennent chacune `m` caractères ('0' ou '1')

## Format de la sortie
Afficher le nombre d'îles.

## Contraintes
- 1 ≤ n, m ≤ 300
- Time limit: 2 secondes
- Memory limit: 256 MB

## Exemple d'entrée
4 5
11110
11010
11000
00000

## Exemple de sortie
1
